# 🎵 Chorus — Many minds, one answer.

> An AI deliberation product that makes ChatGPT, Claude, Gemini, Perplexity, Grok & DeepSeek debate each other — and delivers one synthesized, verified answer.

---

## 🧠 The Problem

Power users today juggle **6 different AI tools** — ChatGPT, Claude, Gemini, Perplexity, Grok, DeepSeek.

- Each model gives a **different answer**
- No way to know **whose answer is correct**
- Users manually copy-paste and triangulate — **wasting 2+ hours/day**

---

## 💡 The Solution

**Chorus** is an AI orchestration layer that:

1. Takes **one query** from the user
2. Routes it to **6 frontier models simultaneously**
3. Makes the models **debate and challenge** each other
4. Synthesizes **one high-confidence answer** with a consensus score

**One query. Six minds. One answer.**

## 🔍 Key Product Decisions

### North Star Metric
**Weekly Active Queries (WAQ)** — number of queries run through Chorus per week per active user.

### MoSCoW Prioritization

**Must Have (V1)**
- Multi-model query routing
- Individual model response display
- Synthesized answer with confidence score
- Query history

**Should Have (V1.5)**
- Debate round transparency
- Model panel customization
- Export synthesis as document
